## 📘 Topics & Concepts Covered
- **CSS Flexbox** — layout for navigation, text content, and image scrolling  
- **Horizontal Scroll** — `overflow-x: auto` and `white-space: nowrap`  
- **Scrollbar Styling** — `::-webkit-scrollbar` & `::-webkit-scrollbar-thumb`  
- **Typography** — headings, text spacing, and font-weight adjustments  
- **Responsive Containers** — used `flex-shrink` and fixed widths for consistent layout  
- **Object-fit & Object-position** — maintain aspect ratio and proper image display  

---

## 🧠 Key Takeaways
- Horizontal scrolling layouts allow showcasing multiple items without vertical scrolling  
- Flexbox makes aligning and spacing elements consistent across sections  
- Custom scrollbars improve the visual experience  
- Clean structure and reusable classes simplify scaling for more sections
